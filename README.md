# Pokemon Battle Revolution Importer for Unity

Pokémon Battle Revolution (PBR) OBM files importer for Unity Engine (5.6.6 in this case to e able to use it for Nintendo 3DS). To use this, just add it to your project. Extract the FSYS with PokemonFSYSConverter.exe from https://github.com/vgmoose/PokemonFSYSConverter to get the TGA and OBM, and use the editor tab to import them all.

This script was a port of the original Python script from Tiberius available in https://archive.vg-resource.com/thread-22105.html
